# RI1

Este projeto é uma atividade acadêmica desenvolvida para fins de estudo e avaliação, focada na implementação de lógica de comparação entre objetos no ecossistema **JavaScript**.

## Sobre o Projeto

O sistema realiza a comparação de atributos e estatísticas entre diferentes personagens do universo **Marvel**.

### Tecnologias Utilizadas
* **Linguagem:** JavaScript
* **Ambiente:** Desenvolvido e testado utilizando VS Code.

## Como Executar

1.  **Clone este repositório:**
    ```bash
    git clone https://github.com/jopaul0/RI1.git
    ```
2.  **Importe o projeto:**
    Abra a pasta do projeto em sua IDE de preferência.
3.  **Execute a classe Principal:**
    Localize o arquivo que contém o método `main` e execute-o para ver as comparações no console. (Necessário ter o Node.js instalado)
